# sikafroLoan
A full-featured fintech loan web application built for the Ghanaian market. No frameworks, no backend — just pure HTML, CSS and JavaScript.

## Live Demo
View Live 🌐 https://ankamahjohnson.github.io/sikafroLoan/

## Features

- **Landing page** — Hero, loan products, testimonials, how it works, CTA
- **Authentication** — Register with OTP phone verification, login, forgot password
- **Loan Application** — 6-step wizard with personal details, income, two guarantors, Ghana Card upload and NIA verification
- **Dashboard** — Total borrowed, outstanding balance, next EMI, credit score
- **My Loans** — Filter by status, loan details modal, PDF statement download
- **Payments** — MTN MoMo, Vodafone Cash, AirtelTigo, bank transfer, payment history
- **Disbursement Flow** — Choose how approved funds are received
- **Eligibility Checker** — Instant pre-qualification based on income and credit score
- **Referral System** — Unique referral codes, earnings tracker, referral history
- **Support Chat** — Keyword-aware live chat with FAQ shortcuts
- **Payment Accounts** — Add MoMo, bank, Visa or GhIPSS accounts
- **Dark Mode** — Full dark theme across all pages including landing
- **Responsive** — Mobile-first, works on all screen sizes
- **Contact Picker** — Pick guarantors directly from phone contacts (Android Chrome)

## Tech Stack

| Layer | Technology |

| Structure | HTML5 |
| Styling | CSS3 |
| Logic | Vanilla JavaScript (ES6+) |
| Charts | Chart.js |
| PDF Export | jsPDF + AutoTable |
| Storage | localStorage / sessionStorage |
| Fonts | Inter (Google Fonts) |
